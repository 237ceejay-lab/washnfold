▸ Automatic Log Email Settings

    
The app can send activity logs to your email automatically every day in the background — no user action needed — using EmailJS (free). If EmailJS is not set up, a fallback log download will trigger every 3.5 days on logout.


    
    

      
☁️ EmailJS Configuration (for automatic daily emails)

      

        1. Create a free account at emailjs.com

        2. Add an Email Service (Gmail recommended) → copy your Service ID

        3. Create a Template with variables {{to_email}}, {{subject}}, {{message}} → copy your Template ID

        4. Copy your Public Key from Account → API Keys

        5. Paste all three below and click Save.
